# Images for the docs

These files are referenced by the README and the OAuth guide. The
`architecture.svg` diagram is already included. The rest are **screenshots you
capture yourself** — the written steps work without them, but images make the
guide much friendlier.

Drop each file here with the exact filename below.

## For the README
- [ ] `dashboard.png` — a photo or screenshot of your running wall display.

## For `google-oauth-setup.md`
- [ ] `oauth-01-create-project.png` — the New Project dialog in Google Cloud Console.
- [ ] `oauth-02-enable-calendar-api.png` — the Google Calendar API page with the **Enable** button.
- [ ] `oauth-03-consent-screen.png` — the OAuth consent screen with app name filled in.
- [ ] `oauth-04-create-client.png` — Create OAuth client ID, **Web application** selected.
- [ ] `oauth-05-authorized-origins.png` — the **Authorized JavaScript origins** section with a URL added.
- [ ] `oauth-06-client-credentials.png` — the dialog showing the generated Client ID and Secret (blur the secret before sharing publicly).

Tip: PNG at roughly 1200px wide looks crisp without bloating the repo. Crop to
just the relevant panel rather than the whole browser window.
